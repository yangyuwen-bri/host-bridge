# 项目导入模板（复制后按条目替换）

将下面模板粘贴到项目「小说文本」输入区，然后把 `【原文】` 部分替换为你选中的条目原文。

```text
【素材来源】
《子不语》卷一（公有领域）

【条目标识】
zby-v01-008

【标题】
骷髅报仇

【改编目标】
做成强冲突、快节奏、结尾反转的短视频，时长 30~60 秒。

【分镜偏好】
1. 开场 3 秒给冲突钩子
2. 每镜头台词尽量短
3. 结尾必须有记忆点

【原文】
（粘贴你选择的《子不语》条目原文）
```

## 建议参数

- 首轮文本长度：500~5000 字
- 每条先出 1 版，确认画风后再做第 2 版
- 首轮不要开口型同步，优先拿到可发版本
