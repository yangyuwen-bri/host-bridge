# 《子不语》文本素材库（首版）

这个素材库用于你当前项目的「快速出片」目标，先提供可直接选题的短篇目录，不等 API 配置完成也能先做内容策划。

## 文件说明

- `zibuyu_story_catalog.csv`：短篇条目目录（爆点标签、建议分镜规模、推荐输入字数）
- `top8-first-batch.md`：优先测试的 8 条（按“最快出成片”排序）
- `project_input_template.md`：粘贴到项目配置页时的文本模板

## 使用方式（建议）

1. 先从 `top8-first-batch.md` 里选 1 条，做首个验证片。
2. 在应用里把单条文本控制在 5,000~20,000 字（更稳）。
3. 先跑到脚本/分镜，确认风格后再跑视频与配音。
4. 第一轮先不开口型同步，优先拿到可发布成片。
